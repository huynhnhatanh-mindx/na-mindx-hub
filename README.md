# na-mindx-hub

Dự án ban đầu được thiết lập với cấu trúc Frontend và Backend riêng biệt.

## Cấu trúc thư mục

- `frontend/`: Ứng dụng Frontend viết bằng **React, Vite, TypeScript** và **Vanilla CSS**.
- `backend/`: Ứng dụng Backend viết bằng **Node.js, Express** và **TypeScript**.

## Yêu cầu hệ thống

- **Node.js**: Phiên bản 18+ được khuyến nghị.
- **npm** hoặc **yarn** / **pnpm**.

## Hướng dẫn Khởi chạy nhanh

### 1. Khởi chạy Backend

Di chuyển vào thư mục backend và chạy lệnh cài đặt dependencies, sau đó chạy môi trường phát triển:

```bash
cd backend
npm install
npm run dev
```

Server backend sẽ mặc định chạy tại: `http://localhost:5000`

### 2. Khởi chạy Frontend

Di chuyển vào thư mục frontend và chạy lệnh cài đặt dependencies, sau đó chạy môi trường phát triển:

```bash
cd frontend
npm install
npm run dev
```

Ứng dụng frontend sẽ chạy tại: `http://localhost:5173` (hoặc cổng khác tùy thuộc vào cổng còn trống).
